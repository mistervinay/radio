🎧 Live Radio Player (Hindi Channels)

A free live radio streaming web app that lets users listen to popular Hindi radio stations anytime, anywhere — just like a mobile radio app 📱

🚀 Features
📻 Live Hindi Radio Channels
⚡ Fast streaming & lightweight player
📱 Mobile App-like UI
🎵 Background play support
🔊 Volume & play/pause controls
🌐 Works on all browsers
❤️ Simple & clean design
📡 Available Channels (Example)
Radio Mirchi
Red FM
Big FM
Vividh Bharati
AIR FM Gold

(You can add more channels anytime)

🛠️ Tech Stack
HTML5
CSS3
JavaScript
Audio Streaming APIs

🔧 Installation
git clone https://github.com/your-username/live-radio-player.git
cd live-radio-player
open index.html
